Bike Sales Prediction

Developed an end-to-end Machine Learning project to predict bike sales and profitability
using historical sales data from Europe and America.

Key Features
Data Cleaning and Preprocessing
Exploratory Data Analysis (EDA)
Feature Engineering
Sales and Profit Prediction
Data Visualization
Model Evaluation and Comparison
Technologies Used
Python
Pandas
NumPy
Matplotlib
Seaborn
Scikit-learn
Jupyter Notebook
Results

Built and evaluated multiple regression models, including Polynomial Regression and Decision Tree Regressor,
to forecast bike sales and profit trends. Decision Tree Regressor achieved the best performance for prediction.
